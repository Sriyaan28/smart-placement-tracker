### SMART PLACEMENT TRACKER

# ROLES:

- Students
- Mentors(Trainers and Guidance)
- Companies
- Admin

# FEATURES:

 # GENERAL:
  - All roles can register (except ADMIN - admin credentials will be provided)
  - All roles should be able to login and logout.
  - Role based access control
  - View Profile
  - Update profile (name, bio ,password)
  - Delete their profile (using password for verification)

 # STUDENT:

 - CRUD operations on Resume
 - View company profiles
 - View job openings
 - View job statistics
 - Apply for jobs
 - View application status
 - View job details
 - View mentor's profiles
 - View training programs
 - Apply for training programs
 - View events
 - Apply for events
 - View guidance posts
 - Receive notifications (job updates, event updates, mentor updates)
 - Chat with mentors
  
 # MENTORS:

 - CRUD operations on training programs
 - CRUD operations on events
 - Create posts for guidance
 - View students profiles
 - Manage student applications
 - Chat with students
 - Receive notifications (application updates, training updates, event updates)

 # COMPANIES:

 - CRUD operations on job openings
 - View student profiles
 - Manage student applications
 - Create interview schedules for applied students
 - Receive notifications (applications received, interview updates)

 # ADMIN:

 - Manage users (add, remove, update users)
 - Manage roles (add, remove, update roles)
 - Manage companies (add, remove, update companies)
 - Manage job openings (add, remove, update job openings)
 - Manage training programs (add, remove, update training programs)
 - Manage events (add, remove, update events)
 - Manage guidance posts (add, remove, update guidance posts)
